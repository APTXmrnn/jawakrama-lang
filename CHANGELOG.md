# Catatan Perubahan (Changelog)

Kabeh owah-owahan penting ing proyek iki bakal dicathet ing file iki.

## [1.1.0] - 2025-11-29

### Ditambahake
- **Standard Library**: Modul gawan `Wicalan` (Matematika), `Serat` (String), lan `Wekdal` (Wektu).
- **REPL**: Mode interaktif kanthi nglakokake printah `jawa` tanpa argumen.
- **Source Maps**: Dhukungan debugging kanggo nuduhake baris kode asli ing file `.jwa`.
- **Array Methods**: Keyword anyar `peta` (map), `saring` (filter), `uda` (reduce), `wonten_ing` (includes), `gabung` (join), `pados` (find).
- **Error Handling**: Pesen error sing wis diterjemahake menyang Basa Jawa Krama.
- **Dokumentasi**: Nambahake file `CONTRIBUTING.md`, `API.md`, lan nganyari `TUTORIAL.md`.
- **Contoh**: File conto anyar ing folder `contoh/` (`contoh_data.jwa`, `contoh_pustaka.jwa`).

## [1.0.0] - 2025-11-26

### Ditambahake
- Rilis pisanan Jawakrama.
- Transpiler dhasar saka Jawakrama menyang JavaScript.
- CLI `jawakrama` kanggo mbukak file `.jwa`.
- Dhukungan kanggo variabel, kontrol alur (if, for, while), fungsi, lan kelas.
