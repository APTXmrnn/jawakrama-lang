# Panduan Kontribusi Jawakrama

Matur nuwun sampun kersa mbiyantu ngembangaken Jawakrama! Menika wonten sawetawis paugeran kangge kontribusi.

## Carane Kontribusi

1.  **Fork** repositori iki.
2.  Gawe **branch** anyar kanggo fitur utawa perbaikan (`git checkout -b fitur-anyar`).
3.  Tindakake owah-owahan lan **commit** (`git commit -m 'Nambah fitur anyar'`).
4.  **Push** menyang branch sampeyan (`git push origin fitur-anyar`).
5.  Gawe **Pull Request**.

## Pelaporan Masalah (Bug Reporting)

Menawi manggihaken bug, sumangga damel *issue* anyar kanthi njlentrehaken:
- Langkah-langkah kangge nreproduksi bug.
- Apa sing dikarepake vs apa sing kedadeyan.
- Versi Jawakrama sing digunakake.

## Gaya Penulisan Kode (Code Style)

- Gunakake indentasi 2 spasi utawa 4 spasi (konsisten).
- Wenehi komentar ing bagean kode sing rumit.
- Gunakake jeneng variabel sing jelas.

Matur nuwun!
